🔑 Key Modules
config/packs.ts
Fonte da verdade para requisitos de cada pack. Exporta PACK_CONFIGS — mapa de PackType+DeckSize → número de slots por role. A UI e a validação lêm daqui.
config/deck.ts
Define os arrays de suits e ranks para cada deck size. DECK_40 e DECK_54. Gera todos os cards (suit+rank) de um deck.
lib/card-gen/get-photo-for-card.ts
Função pura: dado Card + PackType + OrderPhoto[], devolve o OrderPhoto correto (ou null para template padrão). Coração da lógica de mapeamento.
lib/card-gen/generate-svg-overlay.ts
Gera SVG string com rank (top-left + inverted bottom-right), suit symbol, e card border. Estilo hand-drawn/doodle para match com o logo.
lib/card-gen/compose-card.ts
Sharp pipeline: download do Storage → resize 825×1125 → composite SVG buffer → PNG output → upload ao Storage. Uma carta de cada vez.
lib/card-gen/deck-builder.ts
Orquestra a geração de todas as cartas de um order em paralelo (Promise.all por batch). Escreve os order_cards na DB.
